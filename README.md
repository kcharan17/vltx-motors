## 🚗 About

VLTX Motors is a concept landing page for the **Model X1** — an electric supercar with 847 km range, 680 HP, and a 2.4s 0–100 km/h sprint. The page is designed to feel like a premium automotive brand website with dramatic visuals and smooth interactions.

---

## ✨ Features

- **Cinematic hero section** — full-viewport car reveal with headlight glow animation
- **Scroll-driven assembly sequence** — sticky scroll section with phase text transitions
- **Animated performance counters** — numbers count up with easing when scrolled into view
- **3D mouse-tilt showcase** — perspective tilt effect on the car image based on cursor position
- **Fixed navigation** — transparent header that frosts on scroll
- **Fully self-contained** — car image embedded as base64, zero external image dependencies
- **Responsive** — works on mobile and desktop

---

## 🛠️ Built With

- HTML5
- CSS3 (custom properties, scroll-snap, sticky positioning, keyframe animations)
- Vanilla JavaScript (IntersectionObserver, requestAnimationFrame, scroll events)
- Google Fonts — Rajdhani + Inter

## 🎨 Design Decisions

- **Dark palette** (`#050507` background) to let the crimson car pop
- **Rajdhani** for headings — angular, mechanical, automotive feel
- **Scroll-snap** for section-by-section storytelling
- **No frameworks** — keeps it fast, portable, and easy to understand
